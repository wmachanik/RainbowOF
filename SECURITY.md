# Security Policy

## Supported Versions

Curent versions of projects are all in pre-alph stage. Nothing is supported yet.

No login in the app is currently supported.


| Version | Supported          |
| ------- | ------------------ |
| 0.0.0.x | :nothing:          |
| < 0.0   | :x:                |
