Rainbow Order Fulfilment System Read Me
---------------------------------------------------------------------

Rainbow OF is a Rainbow Order Fulfilment that is an open source solution to managing order fulfilment from WooCommerce to the customer.

Designed to pull data from Woo, or import from a legacy system it handles the order from placement to fulfilment. Minus the accounting.


Coding standards:
------------------

- Local class variables: "_" prefix with camel case similar to the class name. The exception is the model with will prefix the Pascal case with the word model
- Local function/routine variables: "_" prefix with Pascal case.
- Parameters: where appropriate use a word prefix to explain the parameter action (like delete or update or new) or use the prefix "p" but make sure you add a comment about using ///

enums: to be in camel case.
